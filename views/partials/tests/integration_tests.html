Setting up your integration tests with Protractor

  • Protractor is the E2E framework test for Angular
  • Built on WebdriverJS (aka Selenium Server)
  • Has it's own runner, independant of karma
  • A lot slower than karma, because you have to serve your whole application and interact with it
  • As with karma, the specific behavior if defined through a configuration file (protractor.conf.js by default)
